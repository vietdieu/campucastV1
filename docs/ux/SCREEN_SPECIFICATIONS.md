# Screen Specifications

This document defines the specifications for core screens.

## 1. Home
- **Purpose**: Immediate work resumption and status monitoring.
- **Priority**: Resume Working > Running Tasks > Recent Projects.
- **Components**: Hero Card, Recent Activity Card, Task Monitor.

## 2. Create
- **Purpose**: Guided generation of commute content.
- **Priority**: Task Sequence (RSS -> AI -> Voice -> Preview).
- **Components**: Workflow Panel, Configuration Panel, Preview Panel.

## 3. Library
- **Purpose**: Project and asset management.
- **Priority**: Project Management > Inspection.
- **Components**: Project List, Inspector Panel.
