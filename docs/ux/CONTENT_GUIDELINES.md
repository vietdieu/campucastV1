# Content Guidelines

Chuẩn hóa ngôn ngữ và thông điệp hướng tới người vận hành (Operator), loại bỏ thuật ngữ kỹ thuật.

## 1. Nguyên tắc chung
- Sử dụng góc nhìn người dùng: "Tạo", "Chỉnh sửa", "Xuất bản" thay vì "Compile", "Hydrate", "RSS Parser".
- Ngắn gọn, súc tích, dễ hiểu.

## 2. Quy ước từ ngữ
| Thuật ngữ cũ (Tech) | Thuật ngữ mới (Operator) |
| :--- | :--- |
| Compile | Đang tạo... |
| Hydrating State | Đang khôi phục... |
| Queue | Hàng đợi xử lý |
| Workspace | Dự án |
| RSS/AI Host | Công cụ tạo nội dung |

## 3. Thông điệp lỗi
- **Thay vì**: "Error: Audio file not generated."
- **Nên dùng**: "Chưa tạo được bản ghi âm. [Thử lại] [Đổi giọng]"

## 4. Trạng thái (Microcopy)
- **Đang tạo**: "Đang tạo bản tin...", "Đang render video..."
- **Thành công**: "Đã tạo thành công", "Sẵn sàng"
- **Thất bại**: "Không thực hiện được", "Cần chú ý"
