# Design Grammar - CommuteCast v3.2

Standardizing UI components to ensure visual consistency.

## 1. Component Grammar
- **Card**: Header (Icon/Title), Body (Content), Footer (Metrics), Actions (Primary/Secondary).
- **Button (CTA Hierarchy)**:
    - Primary: High-contrast, brand-accent.
    - Secondary: Subdued, border-based.
    - Ghost: Transparent, hover-only.
    - Danger: Rose/red accent.
- **Panel/Sidebar**: Surface-subtle, border-subtle.
- **Status Grammar**:
    - Mission: Ready, Running, Paused, Archived, Completed.
    - System: Healthy, Warning, Critical.

## 2. Visual Tokens (Design Tokens)
- Spacing: Standardized 4px grid.
- Radius: app-2xl (16px/24px).
- Typography: Inter (Sans), JetBrains Mono (Data).
- Iconography: Lucide-react (16px base).
