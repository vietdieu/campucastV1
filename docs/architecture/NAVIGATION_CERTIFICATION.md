# Navigation Certification - CommuteCast v3.2

This certification freezes the primary navigation structure to exactly 4 Workstations. No additional primary tabs are permitted.

## 1. Primary Navigation (L1) - Sidebar
1.  **🏠 Home**: The command center for active work and status.
2.  **✏️ Create**: The linear pipeline for mission production.
3.  **📚 Library**: The structured storage for all intellectual property.
4.  **⚙️ Settings**: Global platform configuration.

## 2. Secondary Navigation (L2) - Internal Routing

### Create (Linear Pipeline)
- `Create` -> `Source Selection`
- `Create` -> `Script Drafting`
- `Create` -> `Host Selection`
- `Create` -> `Production Status`

### Library (Object Browsing)
- `Library` -> `Projects` (Root)
- `Library` -> `Missions` (per Project)
- `Library` -> `Assets` (per Mission)

### Settings (Tabbed Panels)
- `Settings` -> `General` (Visuals/Theme)
- `Settings` -> `Identity` (Voice/Host)
- `Settings` -> `News` (Global Filter)
- `Settings` -> `Privacy & Advanced`

## 3. Tertiary Navigation (L3) - Command Palette & Search
- **Universal Search (Ctrl+K)**: Cross-references Missions, Assets, and Settings.
- **Contextual Actions**: Buttons within Views that trigger deep-links to other Workstations (e.g., "Edit Source" from the Script step).

## 4. Certification Statement
All existing Tabs (`RSS`, `AI Host`, `Analytics`, `Podcast`) are hereby DEPRECATED as primary navigation items and must be re-homed into the structures above.
