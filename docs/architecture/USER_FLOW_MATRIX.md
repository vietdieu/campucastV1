# User Flow Matrix - CommuteCast v3.2

Mapping Personas to Journeys to ensure Workflow-first UX.

| Persona | Primary Journey | Critical Path |
| :--- | :--- | :--- |
| **First-time Operator** | Home -> New Mission | Home -> Studio -> RSS -> Generate -> Preview -> Publish |
| **Returning Operator** | Home -> Resume Mission | Home -> Resume -> Continue Editing -> Publish |
| **Content Creator** | Studio -> Assets | Studio -> Authoring -> Assets -> Save -> Archive |
| **Reviewer** | History -> Audit | History -> Timeline -> Snapshot Audit |
| **Administrator** | Settings -> Governance | Settings -> AI/Voice/Storage Config |

## Journey Notes
- All flows are Mission-centric.
- "Resume" takes precedence on `Home` for returning users.
- "Publish" flow is optimized for speed (1-click from Studio).
