# Contextual Guidance & Language Table

Stop using Developer-jargon. Speak to the Operator.

## 1. The Translation Table

| Technical Term (AVOID) | Operator Term (USE) | Description |
| :--- | :--- | :--- |
| Gemini / LLM | **AI Intelligence** | The brain generating the scripts. |
| RSS / Atom | **News Sources** | Where your briefings come from. |
| Pitch / Frequency | **Voice Tone** | How deep or sharp the voice sounds. |
| Rate / Latency | **Reading Speed** | How fast the news is delivered. |
| PCM / Buffer | **Audio Quality** | The clarity of the broadcast. |
| Snapshot / DB | **Save Point** | A moment in time you can return to. |
| Telemetry | **App Analytics** | Data used to improve the experience. |

## 2. Benefit-Driven Subtitles
Every setting should explain **Why** it matters:
- *Bad*: "Set the temperature to 0.7."
- *Good*: "AI Creative Level: Higher levels create more engaging scripts, but may be less literal."

## 3. Error Translation
Never show raw stack traces or "API 404".
- *Bad*: "Provider Error 401: Unauthorized."
- *Good*: "AI Authentication Failed: Please check your API key in the Personalization tab."
